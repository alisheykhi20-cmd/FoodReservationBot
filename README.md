# FoodReservationBot

**Backend Spring Boot project for Telegram food reservation bot (snapshot)**

---

## توضیح پروژه
این پروژه یک سرویس رزرو غذا است که از طریق یک **ربات تلگرامی** با کاربران تعامل دارد.  
کاربران می‌توانند فهرست غذاها را ببینند و با لمس دکمه شیشه‌ای سفارش خود را ثبت کنند.

---

## قابلیت‌ها
- نمایش لیست غذاها از دیتابیس
- امکان انتخاب غذا و ثبت سفارش
- فاکتور کاربر با تعداد هر غذا
- دکمه "تمام" برای ثبت نهایی سفارش

---

## تکنولوژی‌ها
- Java 21
- Spring Boot
- MySQL
- Telegram Bot API (Polling)

---

## اجرا
1. پروژه را با IntelliJ IDEA یا هر IDE دیگری باز کنید.
2. دیتابیس MySQL خود را آماده کنید و اطلاعات اتصال را در `application.properties` یا persistence وارد کنید.
3. ربات تلگرام خود را بسازید و **توکن** آن را در کلاس Bot قرار دهید.
4. پروژه را Run کنید. ربات به تلگرام متصل شده و قابل استفاده است.
